## 📌 Sobre o projeto

Implementação de um jogo da velha completo, com controle de jogadas, validação e verificação automática de vitória.

---

## ⚙️ Funcionalidades

✔ Alternância entre jogadores  
✔ Validação de jogadas  
✔ Atualização do tabuleiro  
✔ Verificação de vitória  
✔ Loop até final do jogo  

---

## 🧠 Conceitos aplicados

- Matrizes  
- Estruturas condicionais  
- Estruturas de repetição  
- Lógica de validação  
- Controle de fluxo  

---

## 🎯 Objetivo

Aplicar lógica de programação em um cenário interativo, simulando o funcionamento completo de um jogo.

---

## 🚀 Aprendizados

Esse projeto reforçou minha capacidade de trabalhar com estruturas mais complexas e lógica sequencial.
